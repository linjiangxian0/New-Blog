发表于 2015-11-19 | 分类于 Leiningen + Clojure之旅

Leiningen简单介绍

Leiningen是Clojure（貌似需要自备梯子）的项目生命周期管理工具，就像Maven在Java中的地位一样。

关于Leiningen具体的情况和使用方法，它的官网和GitHub看看，上面会有更清楚的描述。

安装过程

先说说，基本安装流程，咱们就遇坑填坑。
